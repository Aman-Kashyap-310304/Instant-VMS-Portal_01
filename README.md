# Visitor Management System (VMS) Front-End

This is the front-end implementation of the Visitor Management System. 

## Tech Stack
- **HTML/CSS/JS**: Used for the Landing page, Visitor Register multi-step form, and all other portal pages.
- **React**: Used as an isolated component for the Login page. 

## Project Structure
- `index.html`: Landing / Portal Selector page
- `register.html`: Visitor Register multi-step form page
- `login.html`: Login page (mounts React component)
- `css/`: Stylesheets (`landing.css`, `register.css`, `login.css`)
- `js/`: Scripts (`register.js`, `login.jsx`)

## How to run
You can use `npx serve .` in the project root to run a local web server and open `http://localhost:3000` in your browser.
